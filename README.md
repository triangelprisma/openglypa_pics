# Openglypa

Telegram-бот, генерирующий контент на основе сообщений других пользователей в чате, используя цепь Маркова

### Зачем мне нужно такое, если есть Сглыпа?

Иметь собственное, полностью доступное общественности решение, в котором пользователь знает где хранятся его данные чата и в котором отсутствует навязчивая реклама и paywall лучше! Также, у Сглыпы очень сомнительное пользовательское соглашение, которое показывает, что у автора есть беспрепятственный доступ к вашим данным и запрет на добавление бота "в чаты, в которых распространяется контент, запрещённый законодательством Российской Федерации".
